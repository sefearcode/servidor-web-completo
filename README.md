# Servidor Web en Node.js

Servidor web desarrollado en **Node.js puro (sin Express)**.  
Renderiza HTML dinámico, sirve archivos estáticos (CSS y JS) y maneja rutas básicas.

---

## Características

- Servidor HTTP con Node.js
- Sistema de rutas propio
- Templates HTML con layout
- Archivos estáticos (CSS y JavaScript)
- Listado y detalle de productos
- Página 404 personalizada

---

## Estructura

public/ Archivos estáticos
views/ Templates HTML
data/ Datos de ejemplo
server.js Servidor principal
router.js Rutas
static-server.js Archivos estáticos
templates.js Renderizado HTML
